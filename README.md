plugin
======

C++ Plugin System 

### Build

1. Download the source code from GITHUB

2. Open projects/vs2013/plugin.sln and compile (both DEBUG and RELEASE)

3. Open demo/demo.sln and compile/run

(More project types will be added in the future)
